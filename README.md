# AI-Based Movie Recommendation System 🎬

A beginner-level AI-powered movie recommendation system built with Python. The application recommends movies to users based on genre preferences, age restrictions, ratings, and user interests using core Python programming concepts.

---

## 🚀 Features

- Personalized movie recommendations
- Genre-based filtering
- Age restriction filtering
- Rating-based recommendations
- Mood/interest-based suggestions
- Menu-driven interaction
- Clean and beginner-friendly interface

---

## 🛠 Technologies Used

- Python
- Lists & Tuples
- Conditional Statements
- Loops
- Recommendation Logic

---

## 📊 Functionalities

- Welcome Interface
- Movie Dataset Management
- User Input Collection
- Recommendation Engine
- Rating Prioritization
- Age-Based Filtering
- Recommendation Display

---

## ▶️ How to Run

```bash
python movie_recommendation_system.py
```

Or run the notebook in Google Colab.
